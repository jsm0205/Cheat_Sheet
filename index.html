CSS Selectors and Specificity

Introduction

CSS selectors are used to target specific HTML elements and apply styles to them. Specificity determines which styles are applied when multiple rules target the same element.

Types of Selectors

Selector Example Description
Element Selector p Targets all elements of a specific type (e.g., all paragraph elements).
Class Selector .highlight Targets elements with a specific class attribute (e.g., elements with the class "highlight").
ID Selector #main Targets a single element with a specific ID attribute (e.g., the element with the ID "main").
Attribute Selector a[href] Targets elements with a specific attribute (e.g., anchor elements with an href attribute).
Pseudo-class Selector :hover Targets elements in a specific state (e.g., when the mouse hovers over an element).
Descendant Selector div p Targets all paragraph elements that are descendants of a div element.
Child Selector div > p Targets all paragraph elements that are direct children of a div element.
Adjacent Sibling Selector h1 + p Targets the first paragraph element that is immediately after an h1 element.
General Sibling Selector h1 ~ p Targets all paragraph elements that are siblings of an h1 element.

Specificity

Specificity determines which styles are applied when multiple rules target the same element. The more specific a selector is, the higher its priority.

Specificity Level Description
Inline Styles Styles defined within the HTML element (highest priority).
ID Selectors Selectors using the # symbol (second highest priority).
Class Selectors, Attribute Selectors, Pseudo-class Selectors Selectors using the . symbol or square brackets (third highest priority).
Element Selectors Selectors targeting element types (lowest priority).

Example

Consider these CSS rules:

    p {
      color: blue;
    }

    .highlight {
      color: red;
    }

    #main p {
      color: green;
    }
